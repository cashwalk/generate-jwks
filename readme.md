# Simple JWKS generator 

> JWKS 인증에 사용할 비대칭키를 만들어주는 CLI 패키지입니다.

## Usage

### Serving the app

```sh
$ npx generate-jwks --alg ${alg} --crv ${crv} --output
```

`output`

If present, it will output the generated key to the given path.
